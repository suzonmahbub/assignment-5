Question-1: What are the different ways to select an element in DOM?

There are several ways to select an element form the DOM. 1.getElementById to select element by a specific id. It will return only one element. 2.getElementByClassName used to select all element of a particular class. It returns an HTMLCollection which is a list of element. 3.getElementByTagName used to select elements with specific tag names. It returns an HTMLCollection. 4.querySelector it works like a css selector and returns the first matching element. 5.querySelectorAll this is similar to querySelector,but it will return a list of all matching element.

Question-3: What is event delegation in the DOM?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.

Question-4: What is event bubbling in the DOM?

Event bubbling directs an event to its target. It works like this: When an element (like a button) is clicked, an event is directed to the element. If an event handler is set for the element, the event handler is triggered. Then the event "bubbles up" to the elements parent.
